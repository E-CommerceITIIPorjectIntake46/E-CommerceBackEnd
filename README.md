# E-Commerce
This is a E-Commerce Web Application Api built Using **.NET 10**.

## Technologies Used
- **Language**: C# 14
- **Framework**: .NET 10
- **Database**: SQL Server
- **ORM**: Entity Framework Core

## Components
- Products
- Categories

## Archtecture
N-Tier (3 Layers Arch)
  ```
  Data Layer
      |
      V
  Logic Layer 
      |
      V
  Presentaion Layer (Api)
  ```

## Patterns/Packages Used
- Repository Pattern
- UnitOfWork
- Result Pattern
- Microsoft Identity
- Fleunt Validation
